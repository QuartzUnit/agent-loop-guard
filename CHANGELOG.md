# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] - 2026-03-17

### Added
- 4 detection strategies: exact match, fuzzy similarity, cycle detection, stagnation detection
- Sliding window analysis for memory-efficient monitoring
- Action escalation system (log, warn, raise, kill)
- Zero external dependencies
- 78 tests with full coverage
